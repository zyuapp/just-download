# Just Download

Just Download is an Electron desktop download manager with a Chrome extension that can hand off browser downloads to the desktop app.

## Quick start

```bash
npm install
npm run dev:desktop
```

## Build

```bash
npm run build
```

## Chrome extension

```bash
npm run build:extension
```

Then load `apps/chrome-extension/dist` as an unpacked extension in Chrome.
